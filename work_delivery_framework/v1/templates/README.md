# Templates Index

## 1. Purpose

This folder contains ready-to-use Markdown templates for the canonical Work Delivery Framework artifacts.

Use these files when you are drafting an artifact for a real initiative and need a starting form that already reflects the framework's numbering, traceability, and control expectations.

## 2. How to Use This Folder

1. Confirm the artifact is required in `../artifact_catalog.md`.
2. Read the matching artifact specification in `../artifact_specifications/`.
3. Copy the matching template from this folder into your initiative workspace.
4. Replace blanks with explicit content.
5. Record controlled unknowns in the `Open Items Register` instead of leaving uncontrolled placeholders.

Use `initiative_definition_project_brief_template.md` for that content rather than looking for a standalone acceptance-criteria template.
